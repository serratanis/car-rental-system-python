Rent-Car Management System 🚗
A professional, administrative desktop application developed for managing car rental inventory, tracking rental history, and generating performance reports.
📋 Project Overview
The Rent-Car Management System is a robust desktop solution designed to streamline vehicle fleet operations. Built with a focus on usability and administrative control, it allows managers to track active rentals, manage vehicle inventory, and analyze business performance through detailed reports.
🚀 Key Features
•	Administrative Dashboard: Real-time overview of total vehicles, active rentals, and available stock.
•	Fleet Management: Seamlessly add new vehicles with images, track status, and monitor rental history.
•	Rental/Return Operations: Efficiently process check-outs and check-ins with automated calculations.
•	Performance Reports: Detailed analysis of revenue, occupancy rates, and vehicle-based performance.
•	System Admin Panel: Manage security settings, toggle system preferences (auto-backup, notifications), and customize application appearance (Dark/Light mode).
🛠️ Technical Stack
•	Language: Python
•	GUI Framework: customtkinter (for modern, scalable UI components)
•	Data Persistence: JSON-based storage for lightweight and fast data handling
•	Architecture: Modular design with separate modules for UI, services, and data management.
